# LLM 프롬프트 템플릿 — LoL Realtime Coach

> **작성:** 황기수 (AI/ML)
> **Phase:** 0 (PoC) → Phase 1 통합용
> **대상 LLM:** 1순위 Gemini 2.5 Flash (무료 티어), 폴백 OpenAI / Grok
> **출력 언어:** 한국어 (ko)

이 문서는 두 종류의 프롬프트를 정의한다.

| 버전 | 용도 | 입력 | 출력 | 모듈 |
|---|---|---|---|---|
| **A. 간단 버전** | Phase 1 자동 트리거 (위험도 임계 초과 시) | risk_score + 컨텍스트 | 1~2문장 한국어 코칭 | `src/llm_caller.py` (Phase 1) |
| **B. 상세 버전** | 핫키(F9) 수동 호출 시 종합 분석 | 60초 미니맵 추적 JSON + 미니맵 이미지 | 4섹션 구조화 한국어 분석 | `src/gemini_client.py` (현행) |

---

## 공통 원칙

- **절대 금지:** 인사말, 사과, 메타 설명("제가 분석해보니..."), 영어 단어 노출(고유명사 챔피언명 제외).
- **TTS 친화:** 모든 출력은 음성으로 읽힐 가능성이 있으므로 마크다운/이모지/괄호 강조 사용 금지.
- **속도 우선:** Phase 1 자동 트리거는 응답 지연이 게임 플레이를 방해하므로 max_tokens는 보수적으로.
- **컨텍스트 부재 시 환각 금지:** 데이터에 없는 챔피언/오브젝트 언급 금지. 모르면 "정보 부족"으로.

## A. 간단 버전 — Phase 1 자동 트리거용

### A.1 역할 (System)

```
너는 LoL 매크로 코치다. 위험도와 맥락 정보를 받으면 1~2문장의 한국어 음성 코칭을 즉시 출력한다.
규칙:
- 출력은 1~2문장. 총 80자 이내.
- 명령형 또는 권고형 어투. (예: "후퇴해라", "백 잡는 게 안전하다")
- 인사말, 메타 설명, 마크다운, 이모지, 괄호 강조 금지.
- 데이터에 없는 사실은 절대 만들지 말 것.
```

### A.2 사용자 메시지 (User)

```
[위험도] {risk_score}/100 ({risk_level})
[내 챔피언] {my_champion}
[내 위치] {my_lane}
[근접 적] {nearby_enemies}
[시야] {vision_status}
[트리거 사유] {trigger_reason}
```

### A.3 변수 정의

| 변수 | 타입 | 예시 | 필수 |
|---|---|---|---|
| `risk_score` | int 0~100 | `78` | ✅ |
| `risk_level` | enum | `낮음` / `보통` / `높음` / `매우높음` | ✅ |
| `my_champion` | str | `Jinx` | ✅ |
| `my_lane` | enum | `top` / `jungle` / `mid` / `bot` / `support` / `unknown` | ✅ |
| `nearby_enemies` | list[str] | `["Zed", "LeeSin"]` 또는 `없음` | ✅ |
| `vision_status` | enum | `안전` / `시야없음` / `부분시야` | ✅ |
| `trigger_reason` | str(짧음) | `적 정글러 근접` / `2명 이상 갱` / `시야 소실 5초` | ✅ |

### A.4 입출력 예시

**입력 1 — 매우높음**
```
[위험도] 85/100 (매우높음)
[내 챔피언] Jinx
[내 위치] bot
[근접 적] ["Zed", "LeeSin"]
[시야] 시야없음
[트리거 사유] 미드에서 사라진 적 정글러 + 미드 라이너 봇 방향
```
**기대 출력**
```
지금 즉시 타워 밑으로 후퇴해라. Zed와 LeeSin 갱 위험이 매우 크다.
```

**입력 2 — 보통**
```
[위험도] 55/100 (보통)
[내 챔피언] Garen
[내 위치] top
[근접 적] ["Darius"]
[시야] 부분시야
[트리거 사유] 적 정글러 위치 미상 30초
```
**기대 출력**
```
탑 부쉬에 와드 박고 라인 클리어만 해라. 정글러 동선이 안 보인다.
```

### A.5 호출 파라미터 권장값

| 항목 | 값 | 비고 |
|---|---|---|
| `model` | `gemini-2.5-flash` | 무료 티어, 응답속도 최우선 |
| `temperature` | `0.3` | 일관성 우선, 창의성 낮게 |
| `max_output_tokens` | `120` | 80자 한국어 = 약 60~100 토큰, 여유 |
| `timeout_seconds` | `5` | 자동 트리거는 짧게. 실패 시 무시 |
| 호출 쿨타임 | `5초` | 동일 트리거 5초 내 중복 호출 방지 (`risk_analyzer.py` 참고) |

### A.6 실패 처리

- 응답 빈 문자열 → 출력 스킵 (TTS 호출 안 함)
- 80자 초과 → 첫 마침표/줄바꿈에서 잘라 사용
- 영어/마크다운 포함 → 1회 재시도, 그래도 실패면 스킵

## B. 상세 버전 — 수동 호출 (F9) 종합 분석

> **현행:** `src/gemini_client.py` 의 `SYSTEM_PROMPT_KO` 와 동일. 본 문서는 그 정의를 단일 출처로 정리한다.

### B.1 역할 (System Instruction)

```
너는 T1 페이커급의 LoL 매크로 코치다.
입력으로 지난 60초의 미니맵 객체 추적 데이터(JSON)와 현재 미니맵 이미지를 받는다.
다음 형식으로 한국어로 매우 간결하게 답하라:

[위험도] (낮음/보통/높음/매우높음)
[현재 상황] 한 줄
[추천 로테이션] 1~2줄, 구체적 라인/오브젝트 명시
[30초 후 예상] 한 줄

불필요한 인사말이나 설명 금지. 4개 섹션만 출력.
```

### B.2 사용자 메시지 (User)

`parts[0]` (텍스트):
```
지난 60초 미니맵 데이터:
{summary_json}
```

`parts[1]` (이미지, 선택):
```
mime_type: image/jpeg
data: <현재 미니맵 BGR 프레임을 JPEG q=80으로 인코딩한 바이트>
```

### B.3 `summary_json` 스키마 (요약)

```jsonc
{
  "window_seconds": 60,
  "fps": 30,
  "my_team": "blue",
  "my_champion": "Jinx",
  "tracks": [
    {
      "class": "Zed",
      "team": "enemy",
      "first_seen_s": 12.3,
      "last_seen_s": 58.7,
      "trajectory": [[0.5, 0.5], [0.45, 0.48], "..."],
      "current": {"x": 0.42, "y": 0.46}
    }
  ],
  "missing_enemies": ["LeeSin"],
  "objectives": {
    "next_dragon_s": 75,
    "next_baron_s": 240
  }
}
```

### B.4 출력 예시

```
[위험도] 높음
[현재 상황] 미드 자크와 적 정글러가 보이지 않고 봇 라인 푸시 상태가 깊다.
[추천 로테이션] 봇 듀오는 즉시 일점핑하고 와드 한 개 박아라. 미드는 강가 시야 잡아라.
[30초 후 예상] 봇 다이브 또는 드래곤 시야 싸움 가능성이 높다.
```

### B.5 호출 파라미터 (현행 `configs/config.yaml` 기준)

| 항목 | 값 |
|---|---|
| `model` | `gemini-2.5-flash` |
| `temperature` | `0.4` |
| `max_output_tokens` | `600` |
| `timeout_seconds` | `15` |
| 트리거 | 사용자 핫키 F9 (수동) |

### B.6 출력 검증 체크리스트

수신 직후 다음을 검증한다 (`notifier.py` 표시 전):

- 4개 섹션이 모두 존재하는가? (`[위험도]`, `[현재 상황]`, `[추천 로테이션]`, `[30초 후 예상]`)
- 각 섹션이 빈 문자열이 아닌가?
- `[위험도]` 값이 `낮음/보통/높음/매우높음` 중 하나인가?
- 영어 문장(전체)이 섞여 있지 않은가? (챔피언 고유명사는 허용)

검증 실패 시: 원문 그대로 표시 + 로그 경고. 재호출은 하지 않는다(쿨타임 방해).
