# LoL Realtime Coach

LoL 게임 중 미니맵과 상대 정보를 실시간 분석해 위험 상황을 음성으로 코칭해주는 오버레이 앱.

> **상태**: Phase 0 (2026-04-23 ~ 04-24 셋업)
> **외부 마감**: 2026-05-18

## 주요 기능 (MVP)

1. **YOLO 미니맵 탐지** — 실시간 캡처 + 챔피언 아이콘 인식
2. **위험도 산정** — 상대 위치·수·시야 기반 규칙 점수화
3. **LLM 코칭 호출** — 임계치 초과 시 JSON 프롬프트 → LLM → TTS 대본
4. **오버레이 UI + TTS** — 미니맵 옆 클릭 불가 창, 트레이 최소화
5. **챔피언 조합 승률 그래프** — 게임 시작 시 1회 + 다시보기

## 기술 스택

- **언어**: Python
- **UI**: Tkinter 또는 PyQt (Phase 0에서 결정)
- **AI**: YOLOv26 (자체 학습) + LLM API (OpenAI/Gemini/Claude 중 결정 중)
- **TTS**: 미정 (pyttsx3 또는 gTTS)
- **데이터**: Riot API (Live Game)
